¿Como funciona?

En este proyecto se usa un sensor PT-100 acondicionado con un puente de Wheatstone, el voltaje de salida se manda a un ad620 para amplificar los valores de salida, esa salida es enviada a un convertidor analogio digital (ADC0804) para despues leer los bits de salida mediante un arduino UNO que esta programado para convertir los valores digitales leidos a una temperatura equivalente, estos valores de temperatura se envian al monitor Serial que despues es leido con web sockets, y esa informacion es mandada a un servidor creado con express.

¿Como implementarlo?

  1. Armar el circuito.
  2. Conectar y cargar codigo de Arduino.
  3. Iniciar el servidor de express con el comando "node index.js" ubicado en el directorio src.
